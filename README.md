# Project

Step 1 : install all server dependency and then start server 
  * npm install
  * npm start

Step 2 : install all client dependency and start client
  * cd client 
  * npm install
  * npm start

Step 3 : run json server
  * cd client
  * cd src
  * json-server --watch db.json --port 3004
